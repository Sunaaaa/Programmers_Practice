### #sol2

```python 
def solution(s):
    result = 0 
    
    for idx, number in enumerate(s[::-1]):
        
        print(idx)
        print(number)
        
        if number == '-':
            result *= -1
        else :
            result += int(number) * (10 ** idx)

        print(result)

    return int(s)



print(solution("1234"))
print(solution("-1234"))
```
