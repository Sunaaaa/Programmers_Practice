### 가운데 글자 가져오기

```python
def solution(s):

    if len(s)%2==0:
        return s[int(len(s)/2)-1:int(len(s)/2)+1]
    else :
        return s[len(s)//2]

def solution2(s):

    return s[(len(s)-1)//2:len(s)//2+1]


print(solution("abcde"))
print(solution("qwer"))

print(solution2("abcde"))
print(solution2("qwer"))
```
